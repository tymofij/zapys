Zapys LiveJournal Client
=========================

Supports Markdown and typography

Replacements
-------------------

 * proper quotes
 * (c) ⇒ ©
 * (r) ⇒ ®
 * (tm) ⇒ ™
 * ... ⇒ …
 * n-dash and m-dash
 * з*їм ⇒ з’їм
 * LJ:username ⇒ &lt;lj user="username"&gt;

((citation)) gets special treatment

http://example.com becomes hyperlink "example.com"
